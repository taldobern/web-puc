# TDE-01: Funções em JavaScript

## Objetivo

Demonstrar a aplicação de diferentes tipos de funções em JavaScript para realizar operações matemáticas e manipulação de dados, conforme os requisitos da tarefa de Resolução de Problemas Estruturados em Computação.

## Requisitos Atendidos

O arquivo `tde01.js` (ou `tde01_sem_comentarios.js`) implementa as seguintes funcionalidades, utilizando diferentes sintaxes de função:

| Tipo de Função | Exemplo no Código | Uso |
| :--- | :--- | :--- |
| **Função Nomeada** | `somar`, `subtrair`, `calcularMedia` | Operações básicas e manipulação de array. |
| **Função Anônima** | `dividir` | Operação básica (atribuída a uma constante). |
| **Arrow Function** | `multiplicar`, `filtrarPares` | Operações básicas e manipulação de array (sintaxe concisa). |
| **Função com Callback** | `executarOperacao` | Recebe outra função (`somar`, `multiplicar`, etc.) como parâmetro para execução dinâmica. |
